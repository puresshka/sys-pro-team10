# Часть 3: Node.js API (Вася)

## Что я сделал
Я создал простой HTTP-сервер на Node.js, который:
- Запускается на порту 3000,
- Отдаёт HTML-страницу с надписью "Привет! Это Вася — часть 3 (Node.js)",
- Упакован в Docker-образ,
- Доступен через общий Ingress по пути `/vasya`.

## Файлы в папке `vasya/`
- `server.js` — код сервера,
- `Dockerfile` — инструкция для сборки образа,
- `deployment.yaml` — описание сервиса для Kubernetes.

## Как это работает в системе
1. Minikube запускает Kubernetes.
2. Docker собирает мой образ локально.
3. Kubernetes запускает под из этого образа.
4. Ingress направляет запросы на `/vasya` к моему сервису.
